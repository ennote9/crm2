# ui_v1

New UI layer. Structure:

- **theme/** — tokens, density, light/dark themes
- **app_shell/** — layout container, sidebar, topbar
- **components/** — data_grid, toolbar, chips, dialogs, forms
- **pages/** — orders, order_details, picking, packing, settings
- **utils/** — helpers

Import: `package:crm2/ui_v1.dart` (or relative from lib).
