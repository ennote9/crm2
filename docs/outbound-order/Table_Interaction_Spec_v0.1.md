# Table Interaction Spec v0.1

Применяется к: Orders List, Lines, Pick Tasks, Handling Units/Contents.

## 1) Ширина/компоновка
- Таблица занимает всю доступную ширину контентной области.
- Горизонтальный скролл — только если реально не помещается по минимальным ширинам колонок.
- Sticky header/filters допустимы для сохранения контекста.

## 2) Поиск и фильтры
- Quick Search (одно поле) по ключевым полям (order_no, sku, name).
- Enter применяет, Esc очищает.
- Активные фильтры всегда видимы.

## 3) Сортировка
- Клик по заголовку: asc/desc, индикатор направления.
- Дефолт Orders List: created_at desc.

## 4) Колонки и представления
- Show/Hide, reorder, resize.
- Saved Views (MVP минимум): All / On Hold / Shortage / Today(24h).

## 5) Multi-select и массовые действия (опционально)
- Multi-select (checkbox/shift), action bar.
- Доступно только при наличии прав.

## 6) Клавиатура
- Ctrl/Cmd+F фокус в поиск.
- Enter открыть/применить.
- Видимый фокус, без перекрытий.

## 7) Состояния таблицы
- Loading: без “прыжков”.
- Empty: “почему пусто” + “сбросить фильтры”.
- Error: код/текст + “повторить”.
