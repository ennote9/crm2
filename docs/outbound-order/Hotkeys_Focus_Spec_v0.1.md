# Hotkeys & Focus Spec v0.1 (Desktop-first)

## Глобально
- Ctrl/Cmd+F: поиск
- Esc: закрыть модал/очистить поиск/выйти из режима
- Enter: подтвердить/открыть выделенную строку
- Tab/Shift+Tab: навигация по элементам
- Ctrl/Cmd+Enter: выполнить “Next Step” (где применимо)

## Orders List
- ↑/↓ выбор строк, Enter открыть
- Ctrl/Cmd+F поиск
- Enter применяет фильтр, Esc очищает

## Order Details
- Ctrl/Cmd+Enter: Next Step
- Ctrl/Cmd+L Lines, Ctrl/Cmd+T Pick Tasks, Ctrl/Cmd+U Handling Units, Ctrl/Cmd+E Events
- Ctrl/Cmd+H Hold/Unhold (если доступно)

## Pick Task
- Фокус на qty_picked
- Если недобор — фокус на reason_code
- Alt+1..3: быстрые причины NOT_FOUND/DAMAGED/COUNT_MISMATCH

## Packing
- Ctrl/Cmd+N: новый HU (box)
- Ctrl/Cmd+Enter: Complete Packing (если доступно)

## Ship
- Ctrl/Cmd+Enter: Ship
- W_SHP_001: Enter подтвердить, Esc отменить
