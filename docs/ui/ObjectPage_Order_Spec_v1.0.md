# Object Page Order Spec v1.0 — Outbound Order

## Desktop structure
- Sticky header: order_no + status + signals + Next Step + secondary actions
- Summary strip: aggregates + short
- Tabs: Lines/Reservations/Pick Tasks/HU/Events

## Key rules
- Next Step by available_actions/disabled_actions only
- Disabled reasons displayed (E_*)

## Mobile
- Details screen with segmented tabs/sections
- Next Step prominent (option sticky bottom)
