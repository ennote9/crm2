# Orders Stats Spec v1.0 — Desktop Panel + Mobile Screen

## Follow filters
Stats always reflect current worklist filters/views.

## KPI (MVP)
- Total
- In progress
- On Hold
- Shortage (status Shortage or short>0)
- Shipped in period
- Optional: aging risk

## Visualizations
- Status distribution (MVP)
- Optional trend (7 days)

## Performance
Separate request, debounce, cache TTL.
