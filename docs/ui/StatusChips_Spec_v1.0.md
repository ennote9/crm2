# Status Chips Spec v1.0 — Modern Dynamics Dense

## Rules
- Text is mandatory (color is secondary).
- Compact pill chips.
- Not clickable by default.

## Semantic groups
Neutral: Draft
Info: Released/Allocated/Picked/Packed
In progress: Allocating/Picking/Packing
Warning: Partial Alloc, Shortage
Blocked: On Hold
Success: Shipped/Closed
Danger: Cancelled

## Priority
On Hold > Cancelled > Shortage > In progress > Info/Neutral > Success

## Usage
- Orders list: main status chip
- Object header: always
