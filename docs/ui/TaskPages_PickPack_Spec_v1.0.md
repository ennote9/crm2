# Task Pages Pick/Pack Spec v1.0

## Picking (desktop)
- Left tasks list + right workspace
- Focus qty_picked
- Short pick → reason required + quick reasons (Alt+1..3)
- Flow: complete → next

## Packing (desktop)
- Left HU list + right workspace
- Unassigned remaining always visible
- Complete packing disabled with reason when unassigned>0

## Mobile
- Picking flow (one task per screen)
- Packing flow (HU list → HU detail)
