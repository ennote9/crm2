# Filters + Chips Spec v1.0 — Modern Dynamics Dense

## Components
Filters button + panel + applied chips + clear all.

## Applied chips
- Show Field: Value
- Remove individual chip (×)
- Overflow: +N opens popover listing all chips

## Advanced panel (desktop)
- Popover with sections
- Explicit Apply (recommended)
- Esc closes and discards un-applied changes

## Orders MVP filters
- Status (multi)
- Warehouse
- Date range (Today/This week/Custom)
- Flags: On Hold, Shortage
- Search is also represented as a chip when active

## Mobile
- Bottom sheet for filters
- Chips row optional/compact

## Acceptance
- Chips always reflect applied state
- Clear all resets filters
- Explicit apply works
